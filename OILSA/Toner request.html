COTG Printer - Requesting Service /Toner - OILSA

The only time users should contact COTG directly is to order toner. Toner can also be ordered on the intranet using the same process described below to report issues.

If printer service is needed either call COTG (Chicago Office Technology Group) or complete the below online form.

If the customer needs a replacement printer, upgrade the problem ticket to SEV2 and contact the Hardware on-call.

Phone:

1.Call 630-771-8200 opt.2

2.Provide:

• COTG Printer Tag Number.

• Problem reported by User.

• User Contact Information

• Location/Department.

• Any Troubleshooting already done.

3.Create a SEV3 Problem ticket and assign to Printer Hardware Support- OILSA with the information provided to COTG.

Online:

1. Go to the SAH Intranet (//intranet)>Common Applications.

2. Go to COTG Toner Requests.

3. The default request type is for Toner, if needed change to Service Request.

4. Complete the form with the available information..

5. Create a SEV3 problem ticket and assign to Printer Hardware Support- OILSA with the information provided to COTG.
